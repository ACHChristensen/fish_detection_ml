# Fish Detection ML

A machine learning project that detects a specific type of fish (mackerel) with ~80% accuracy. Developed as part of my Data Science elective during my Datamatiker program. This project demonstrates practical experience with computer vision, data preprocessing, model training, evaluation, and understanding probabilistic outputs in AI systems.

---

## Project Overview

The goal of this project is to create a model capable of recognizing mackerel from images. The project uses **TensorFlow** for model development and focuses on:

- Data preprocessing and cleaning
- Model training and evaluation
- Probabilistic predictions and understanding model limitations
- Learning how to work with real-world datasets
- Has real-time detection using camera feed

---

## Technologies Used

- **Python** – main programming language
- **TensorFlow** – model training and evaluation
- **NumPy / Pandas** – data manipulation
- **Matplotlib / OpenCV** – visualization and image processing
- Git & GitHub – version control

---

## Model Performance

- Accuracy: ~80% on the test set
- Model is probabilistic, meaning predictions are not always 100% correct
- Emphasis on reliability, preprocessing, and evaluation rather than perfect accuracy

---

## Key Learnings

- How to preprocess and clean image datasets for training
- How to implement and train a simple deep learning model for classification
- Understanding probabilistic AI outputs and limitations
- Preparing models for real-world applications
